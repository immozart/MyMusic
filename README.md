### [MyMusic](https://ismusic.herokuapp.com) -
сервис для поиска музыки и добавления подписки на выход новых альбомов избранных исполнителей.

#### MVP Functionality
* Авторизация
* Поиск исполнителей
* Добавление исполнителей в избранное
